# Paper Revision

## Purpose

Improve academic writing while preserving the author's claims, evidence, and technical meaning.

## When to use

- Use when revising a paper draft for clarity, flow, framing, or concision.
- Use when responding to reviewer comments.
- Use when rewriting sections such as abstract, introduction, related work, methods, results, or conclusion.
- Do not use for factual literature review unless sources are available.

## Workflow

1. Identify the target text and the revision goal.
2. Preserve claims, definitions, notation, and evidence unless the user asks for conceptual changes.
3. Prefer concrete edits over broad advice when the text is available.
4. If reviewer comments are present, map each change to the relevant comment.
5. Keep tone aligned with academic writing rather than marketing language.
